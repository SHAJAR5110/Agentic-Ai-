# Learning Repository: AI Agents & LLM Development

Welcome! This repository is designed to take you from beginner to Advanced level in building AI agents and working with Large Language Models (LLMs). Here's what you'll learn throughout this course.

## 📚 Course Overview

This comprehensive guide covers the fundamentals of building intelligent agents, integrating with APIs, and leveraging modern AI/LLM frameworks to create powerful applications.

---

## 🗓️ Weekly Breakdown

### **Week 1: OpenAI SDK Basics + Build Small Agents**
- ✅ Introduction to the OpenAI SDK
- ✅ Setting up your development environment
- ✅ Authentication and API keys
- ✅ **Tool Use**: Understanding function calling
- ✅ **API Integration**: Making requests to external APIs
- ✅ **Building Agents**: Creating your first simple agent
- ✅ Hands-on projects: Weather agent, Calculator agent

---

## 🎯 Learning Outcomes

By the end of this course, you will be able to:

- **Understand** how modern LLMs work and their capabilities
- **Build** functional AI agents with tool use and API integration
- **Integrate** external APIs and services into your agents
- **Implement** multi-step reasoning workflows

## 📦 Tech Stack

- Python 3.8+
- OpenAI Agents SDK

---

## 🚀 Getting Started

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Set up your `.env` file with API keys
4. Start with Week 1 materials
5. Follow along with code examples and exercises

---


## 💡 Key Concepts You'll Master

- **Tool Use / Function Calling** - How agents decide and use external tools
- **Prompt Engineering** - Crafting effective instructions for LLMs
- **Agent Loops** - The core cycle of reasoning, planning, and execution
- **API Integration** - Connecting your agents to external services
- **Context Management** - Handling token limits and memory
- **Error Recovery** - Building robust, fault-tolerant systems
- **Evaluation Metrics** - Measuring agent performance
